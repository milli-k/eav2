1) Change the name of the `.env.example` to `.env` and place your host / api key inside
   
2) Step 2 - install a virtual environment

```console
python -m venv ./.venv
```

3) Step 3 -- activate your virtual environment

```console
source ./.venv/bin/activate
```

4) Step 4 -- install the dependencies

```console
pip install -r requirements.txt
```

5) Step 5 -- customize the script as needed

6) Step 6 -- Run the script

```console
python generate.py
```
